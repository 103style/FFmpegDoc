# FFmpegDoc

###　笔者环境
```
Ubutun : 16.04
NDK : r15c
```



* 克隆到 `$HOME` 目录，`(e.g.  /home/103style/)`

* 配置`NDK`环境变量 `ANDROID_NDK_ROOT`
  ```
  sudo gedit /etc/profile
  ```

  在`profile`最后新增一下语句,修改`/home/103style/android-ndk-r15c`为你的`NDK`目录
  ```
  export ANDROID_NDK_ROOT=/home/103style/android-ndk-r15c
  ```
